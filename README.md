# MultiSupportRecyclerView
- 支持上拉加载
- 下拉刷新
- 空状态错误状转换
- 添加多个header,footer
- 优雅的实现多条目类型列表
- 配合[itemtouchhelper]:https://github.com/mrme2014/ItemTouchHelper
  来实现非侵入式侧滑菜单，长按拖拽
# 使用方式
```java
dependencies{

        compile 'com.qiaomu.library:multirecyclerview:1.0.3'
}
```
